## 地図の作り方

1. Open Street Mapのサイトに行く
https://openstreetmap.jp

2. 「Open Street Map プロジェクトサイト」をクリックする
https://openstreetmap.org/

3. 地図の位置、大きさを調整する
https://www.openstreetmap.org/#map=17/35.69758/139.75923

4. ブラウザの「ズーム」で90%にする。
